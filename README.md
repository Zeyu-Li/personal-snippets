# VS-Code Personal Snippets

A bunch of my personal snippets for all sorts of languages!

## Languages

* HTML
  * `!main` - boilerplate
* CSS
  * `!t` - transitions
* JavaScript
  * `!p` - `console.log`
  * `!d` - `console.debug`
  * `!e` - `console.error`
  * `!w` - `console.warn`
* TypeScript
  * Same as JavaScript
* Python
  * `!p` - `print`
  * `!main` - boilerplate
* Lua
  * `!p` - `print`
* C
  * `!p` - `printf`
  * `!main` - boilerplate
* C++
  * `!p` - `printf`
  * `!c` - `cout` (with endline at the end)
  * `!sc` - `std::cout` (with endline)
  * `!main` - boilerplate
* C#
  * `!p` - `Console.WriteLine`
* Shell/ShellScript
  * `!p` - `echo `
* PowerShell
  * `!p` - `echo `

## Installing Locally

For Windows users:

1. Go to your user directory, ie. `C:\Users\$userName$`
2. Go to `.vscode\extensions`
3. Paste the entirety of this folder inside
4. Restart VS Code
